## 1. List
### 1.1. Ordered List
  ~~~
  <ol>
		<li> First </li>
		<li> Second </li>
		<li> Third </li>
	</ol>
  ~~~
### 1.2. Unordered List
  ~~~
  <ul>
		<li>First</li>
		<li>Second</li>
		<li>Third</li>
	</ul>
  ~~~
 
## 2. Table
~~~
<table>
		<thead> <!-- Column Names -->
			<th>Name</th> 
			<th>Age</th>
			<th>Dept</th>
		</thead>
		<tr> <!-- Column Rows -->
			<td>KimJihyeok</td>
			<td>26</td>
			<td>CS</td>
		</tr>
		<tr> <!-- Column Rows -->
			<td>HongGilDong</td>
			<td>28</td>
			<td>HIS</td>
		</tr>
</table>
~~~
